# FrontEnd実行方法
1. リポジトリをクローン
```bash
git clone https://github.com/Engineer-Guild-Hackathon/team-37-app.git
cd team-37-app/frontend
```

2. 依存関係のインストール
```bash
npm install
```

3. 開発サーバーを起動
```bash
npm run dev
```

4. ブラウザでアクセス
```bash
http://localhost:3000
```
